## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/VictorJGB/automacert-sistemas
   ```
2. Install Packages
   ```sh
   npm install || pnpm install
   ```
